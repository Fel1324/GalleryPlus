# Gallery Plus 📸

Aplicação web para gerenciamento de álbuns e fotos.

<br>

## ✨ Funcionalidades

- **Gerenciamento de Álbuns**: Crie e organize seus álbuns de fotos.
- **Upload de Fotos**: Faça upload de imagens com suporte a múltiplos formatos.
- **Visualização de Fotos**: Navegue pelas fotos com uma interface elegante.
- **Busca e Filtros**: Encontre rapidamente suas fotos e álbuns com ferramentas de busca.

<br>

## 🛠️ Tecnologias Utilizadas

- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Radix UI**
- **React Router**
- **TanStack Query**
- **React Hook Form + Zod**
- **Axios**

<br>

## 🚀 Instalação

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Fel1324/GalleryPlus.git
   cd GalleryPlus
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor Back-End:**
   ```bash
   npm run dev-server
   ```

4. **Inicie o Front-End:**
   ```bash
   npm run dev
   ```

5. **Acesse a aplicação:**
   - [http://localhost:5173](http://localhost:5173)
